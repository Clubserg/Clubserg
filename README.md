## 🧑‍💻 Sobre mí

Soy estudiante de Ingeniería del Software en la Universidad de Oviedo, con experiencia construyendo aplicaciones full-stack, compiladores, herramientas de ciberseguridad y sistemas distribuidos. Me muevo cómodo tanto en el frontend como en el backend, y disfruto especialmente de los proyectos donde el diseño de arquitectura importa.

## 🚀 Proyectos destacados

### 🏢 [sdi2526-entrega2-n](https://github.com/Clubserg) — Space Booking App
> Aplicación de reserva de espacios full-stack con autenticación JWT y API REST.

- **Stack:** Node.js · Express · MongoDB · React
- **Tests:** Selenium (E2E) · RestAssured (API)
- **Aspectos clave:** CORS, JWT, gestión de roles, CRUD completo de espacios y reservas

---

### 🛡️ EWALA Shield — Anti-Typosquatting Tool
> Herramienta OSINT para detectar dominios fraudulentos dirigida a PYMEs españolas. Presentada en el **EWALA Hackathon 2026**.

- **Stack:** Python · Playwright · Docker
- **Técnicas:** pHash/DCT perceptual hashing, análisis de similitud visual, WHOIS, VirusTotal
- **Destacado:** Detección de similitud visual entre dominios como cuarta señal OSINT

---

### 🎮 YOVI — Plataforma de Juegos Multijugador
> Plataforma de juegos online con partidas PvP en tiempo real y ranking global.

- **Stack:** React · TypeScript · Vite · Node.js · Express · Rust · MariaDB
- **Infraestructura:** Docker Compose · nginx · WebSockets
- **Destacado:** Motor de juego en Rust, matchmaking PvP, historial de partidas, sistema de ranking

---

### 🏗️ sdi2526-entrega1-n — Sistema de Reservas (Spring Boot)
> Aplicación web MVC con control de acceso por roles e internacionalización.

- **Stack:** Spring Boot · Thymeleaf · Spring Security · H2
- **Aspectos clave:** CRUD de espacios, roles ADMIN/USER, i18n ES/EN, soft delete, tests Selenium

---

### 🔤 Compilador DLP (C--)
> Implementación de un compilador para un subconjunto del lenguaje C con análisis semántico y generación de código.

- **Lenguaje:** Java
- **Fases:** AST · Identificación (tabla de símbolos/ámbitos) · Type-checking · Generación de código
- **Patrones:** Composite · Visitor (`ValueCGVisitor`, `AddressCGVisitor`)

---

### 🎵 MusicStoreApp
> Tienda de música con API documentada con Swagger y despliegue en MongoDB Atlas.

- **Stack:** Node.js · Express · MongoDB · Twig
- **Destacado:** Documentación Swagger JSDoc completa de todos los endpoints
